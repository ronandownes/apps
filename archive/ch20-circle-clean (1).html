circle radius tangent
The standard equation of a circle with centre (h, k) and radius r:
(x − h)2 + (y − k)2 = r2

Special Case: h = 0, k = 0

When the centre is at the origin, the equation simplifies:

x2 + y2 = r2

Point on Circle at Origin

If point (x1, y1) lies on a circle centred at origin:

x2 + y2 = |x1|2 + |y1|2
💡 Key insight: Because we square the coordinates, the signs don't matter — the sum of two squares is always positive!

Worked Example 20.1 Active Maths 3 – p.485
The circle c has centre (0, 0) and radius 7 units.

Write down the equation of c.
Solution
(1)
x2 + y2 = r2
Formula (centre at origin)
(2)
x2 + y2 = (7)2
Substitute r = 7
(3)
x2 + y2 = 49
72 = 49
(4)
x2 + y2 = 49
Answer ✓
💡 Remember: The right-hand side is always r2, not r. Here 72 = 49.
Understanding: The equation x2 + y2 = 49 represents all points that are exactly 7 units from the origin. Any point (x, y) satisfying this equation lies on circle c.
