Introduction
circle radius tangent
Standard equation of a circle with centre (h, k) and radius r:
(x − h)2 + (y − k)2 = r2

Special Case: h = 0, k = 0 (Origin)

x2 + y2 = r2

Point on Circle Through Origin

x2 + y2 = |x1|2 + |y1|2
💡 Signs don't matter when squaring — sum of squares is always positive!

The circle c has centre (0, 0) and radius 7 units.
Write down the equation of c.
Solution
(1)
x2 + y2 = r2
Formula (centre at origin)
(2)
x2 + y2 = (7)2
Substitute r = 7
(3)
x2 + y2 = 49
Calculate 7² = 49
x2 + y2 = 49
