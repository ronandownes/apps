Statistics & Probability

1.1 Counting

Students should be able to:

1.1aOFFICIALCount the arrangements of n distinct objects (n!)
1.1aSTUDENTWork out how many ways to arrange n different objects using factorials
1.1bOFFICIALCount the number of ways of arranging r objects from n distinct objects
1.1bSTUDENTCalculate how many ways to choose and arrange r items from n different items

Statistics & Probability

1.2 Concepts of Probability

Students should be able to:

1.2aOFFICIALUse set theory to discuss experiments, outcomes, sample spaces
1.2aSTUDENTUse sets to describe experiments and list all possible outcomes
1.2bOFFICIALDiscuss basic rules of probability (AND/OR, mutually exclusive) using Venn diagrams
1.2bSTUDENTApply AND/OR rules and use Venn diagrams to solve probability problems
1.2cOFFICIALCalculate expected value and understand it need not be an outcome
1.2cSTUDENTFind expected value and know it might not be an actual possible result
1.2dOFFICIALRecognise the role of expected value in decision making and fair games
1.2dSTUDENTUse expected value to make decisions and determine if a game is fair

Statistics & Probability

1.3 Outcomes of Random Processes

Students should be able to:

1.3aOFFICIALFind probability that two independent events both occur
1.3aSTUDENTCalculate P(A and B) when events don't affect each other
1.3bOFFICIALApply understanding of Bernoulli trials
1.3bSTUDENTWork with experiments that have exactly two outcomes (success/failure)
1.3cOFFICIALSolve problems involving up to 3 Bernoulli trials
1.3cSTUDENTSolve problems with up to 3 repeated success/failure experiments
1.3dOFFICIALCalculate probability that the 1st success occurs on the nth trial
1.3dSTUDENTFind the probability of getting your first success on try number n

Statistics & Probability

1.4 Statistical Reasoning

Students should be able to:

1.4aOFFICIALDiscuss populations and samples
1.4aSTUDENTExplain the difference between a whole population and a sample from it
1.4bOFFICIALDecide to what extent conclusions can be generalised
1.4bSTUDENTJudge whether sample results can apply to the whole population
1.4cOFFICIALWork with different types of bivariate data
1.4cSTUDENTAnalyse data involving two variables

Statistics & Probability

1.5 Finding, Collecting & Organising Data

Students should be able to:

1.5aOFFICIALSelect a simple random sample
1.5aSTUDENTChoose a random sample where everyone has equal chance of being picked
1.5bOFFICIALRecognise importance of representativeness
1.5bSTUDENTUnderstand why a sample should reflect the whole population
1.5cOFFICIALDiscuss sample surveys, observational studies, designed experiments
1.5cSTUDENTCompare different methods of collecting data: surveys, observations, experiments
1.5dOFFICIALDesign a plan and collect data
1.5dSTUDENTPlan and carry out your own data collection

Statistics & Probability

1.6a Representing Data Graphically

Students should be able to:

1.6aOFFICIALDescribe sample using appropriate graphical/numerical methods
1.6aSTUDENTChoose the right charts and calculations to describe data
1.6bOFFICIALExplore distribution, symmetry, skewness
1.6bSTUDENTDescribe if data is symmetric or skewed left/right
1.6cOFFICIALCompare data sets using back-to-back stem and leaf plots
1.6cSTUDENTUse back-to-back stem and leaf plots to compare two data sets
1.6dOFFICIALDetermine relationship between variables using scatterplots
1.6dSTUDENTDraw scatter plots to see if two variables are related
1.6eOFFICIALRecognise correlation (−1 to +1) and its meaning
1.6eSTUDENTUnderstand correlation values: −1 (strong negative) to +1 (strong positive)
1.6fOFFICIALMatch correlation coefficients to scatterplots
1.6fSTUDENTLook at a scatter plot and estimate the correlation value
1.6gOFFICIALUnderstand correlation does not imply causality
1.6gSTUDENTKnow that correlation doesn't prove one thing causes another

Statistics & Probability

1.6b Representing Data Numerically

Students should be able to:

1.6hOFFICIALRecognise standard deviation and IQR as measures of variability
1.6hSTUDENTUnderstand that standard deviation and IQR measure how spread out data is
1.6iOFFICIALUse calculator for standard deviation
1.6iSTUDENTFind standard deviation using your calculator
1.6jOFFICIALFind quartiles and IQR
1.6jSTUDENTCalculate Q1, Q2 (median), Q3 and the interquartile range
1.6kOFFICIALUse IQR appropriately
1.6kSTUDENTApply IQR to describe spread and compare data sets
1.6lOFFICIALRecognise outliers
1.6lSTUDENTIdentify values that are unusually far from the rest of the data

Statistics & Probability

1.7 Analysing & Drawing Inferences

Students should be able to:

1.7aOFFICIALRecognise sampling variability
1.7aSTUDENTUnderstand that different samples give different results
1.7bOFFICIALInterpret histograms
1.7bSTUDENTRead and interpret histograms
1.7cOFFICIALMake decisions using empirical rule
1.7cSTUDENTUse the 68-95-99.7 rule for normal distributions
1.7dOFFICIALRecognise hypothesis test concept
1.7dSTUDENTUnderstand the basic idea of testing a claim with data
1.7eOFFICIALCalculate margin of error 1/√n for proportion
1.7eSTUDENTFind margin of error using 1/√n for percentages
1.7fOFFICIALConduct hypothesis test on proportion using margin of error
1.7fSTUDENTTest a claim about a percentage using margin of error

Geometry & Trigonometry

2.1 Synthetic Geometry

Students should be able to:

2.1aOFFICIALPerform constructions 16–21
2.1aSTUDENTComplete constructions 16 to 21 with compass and straight edge
2.1bOFFICIALUse terms: theorem, proof, axiom, corollary, converse, implies
2.1bSTUDENTKnow the meaning of: theorem, proof, axiom, corollary, converse, implies
2.1cOFFICIALInvestigate and use theorems 7, 8, 11, 12, 13, 16, 17, 18, 20, 21 and corollary 6
2.1cSTUDENTApply theorems 7, 8, 11–13, 16–18, 20–21 and corollary 6 to solve problems

Geometry & Trigonometry

2.2 Coordinate Geometry

Students should be able to:

2.2aOFFICIALUse slopes to show lines parallel/perpendicular
2.2aSTUDENTProve lines are parallel (same slope) or perpendicular (slopes multiply to −1)
2.2bOFFICIALRecognise ax + by + c = 0 as linear
2.2bSTUDENTKnow that ax + by + c = 0 is the equation of a straight line
2.2cOFFICIALSolve slope problems
2.2cSTUDENTSolve problems involving slope, equations of lines, and intersections
2.2dOFFICIALCalculate area of a triangle
2.2dSTUDENTFind the area of a triangle using coordinates
2.2eOFFICIALRecognise (x−h)² + (y−k)² = r² as circle equation
2.2eSTUDENTKnow that (x−h)² + (y−k)² = r² is a circle with centre (h, k) and radius r
2.2fOFFICIALSolve problems involving a line and circle centred at origin
2.2fSTUDENTFind where a line meets a circle centred at the origin

Geometry & Trigonometry

2.3 Trigonometry

Students should be able to:

2.3aOFFICIALUse Pythagoras' theorem (2D)
2.3aSTUDENTApply a² + b² = c² to find sides in right-angled triangles
2.3bOFFICIALUse trigonometry to find area of triangle
2.3bSTUDENTFind triangle area using Area = ½ab sin C
2.3cOFFICIALSolve problems using sine and cosine rules (2D)
2.3cSTUDENTUse sine rule and cosine rule to solve triangle problems
2.3dOFFICIALDefine sin θ, cos θ, tan θ for all θ
2.3dSTUDENTUnderstand sin, cos and tan for any angle using the unit circle
2.3eOFFICIALSolve problems involving arc length and sector area
2.3eSTUDENTCalculate arc length and sector area of circles
2.3fOFFICIALWork with trigonometric ratios in surd form
2.3fSTUDENTUse exact values like sin 30° = ½, cos 45° = √2/2

Geometry & Trigonometry

2.4 Enlargements

Students should be able to:

2.4aOFFICIALInvestigate enlargements and effect on area
2.4aSTUDENTUnderstand that enlarging by factor k multiplies area by k²
2.4bOFFICIALIdentify centre of enlargement and scale factor k ∈ ℚ
2.4bSTUDENTFind the centre of enlargement and scale factor from a diagram
2.4cOFFICIALSolve problems involving enlargements
2.4cSTUDENTApply enlargements to solve geometry problems

3.1bSTUDENTDo calculations involving surds and irrational numbers
3.1cOFFICIALRevisit operations in ℕ, ℤ, ℚ, ℝ
3.1cSTUDENTWork confidently with natural numbers, integers, fractions and real numbers
3.1dOFFICIALConsolidate factors, multiples, primes
3.1dSTUDENTFind factors, multiples and identify prime numbers
3.1eOFFICIALExpress numbers in prime factor form
3.1eSTUDENTWrite numbers as products of primes (e.g. 60 = 2² × 3 × 5)
3.1fOFFICIALExpress numbers in standard form a × 10ⁿ
3.1fSTUDENTWrite very large or small numbers in scientific notation

3.2bSTUDENTWrite a formula for the nth term of a sequence

Number

3.2b Indices (Index Laws)

Students should be able to:

3.2eOFFICIALApply aᵖ × aᵍ = aᵖ⁺ᵍ and aᵖ ÷ aᵍ = aᵖ⁻ᵍ
3.2eSTUDENTMultiply powers: add indices. Divide powers: subtract indices
3.2fOFFICIALApply a⁰ = 1 and (aᵖ)ᵍ = aᵖᵍ
3.2fSTUDENTKnow that anything to power 0 is 1; power of a power: multiply indices
3.2gOFFICIALApply a^(1/q) = ᵍ√a and a^(p/q) = ᵍ√(aᵖ)
3.2gSTUDENTConvert between fractional indices and roots: a^(½) = √a
3.2hOFFICIALApply a⁻ᵖ = 1/aᵖ and (ab)ᵖ = aᵖbᵖ
3.2hSTUDENTNegative indices mean reciprocals; distribute powers over products

3.3cOFFICIALCalculate average rates of change

Algebra

4.2aOFFICIALSolve f(x) = g(x) where f, g are linear
4.2aSTUDENTSolve linear equations like 3x + 5 = 2x − 1

Functions

5.1 Functions

Students should be able to:

5.1aOFFICIALRecognise a function assigns unique output to input
5.1aSTUDENTUnderstand that a function gives exactly one output for each input
5.1bOFFICIALForm composite functions
5.1bSTUDENTFind composite functions like f(g(x)) — apply one function then another
5.1cOFFICIALGraph functions: ax + b, ax² + bx + c, ax³ + bx² + cx + d, abˣ
5.1cSTUDENTSketch linear, quadratic, cubic and exponential graphs
5.1dOFFICIALInterpret f(x) = g(x) as comparison of functions
5.1dSTUDENTUnderstand that solving f(x) = g(x) finds where two graphs meet
5.1eOFFICIALUse graphical methods to solve f(x) = 0 or f(x) = k
5.1eSTUDENTRead solutions from a graph: where it crosses the x-axis or a horizontal line
5.1fOFFICIALInvestigate limit of a function
5.1fSTUDENTUnderstand what value a function approaches as x gets very large or small

Functions

5.2 Calculus

Students should be able to:

5.2aOFFICIALFind first and second derivatives of linear/quadratic/cubic functions
5.2aSTUDENTDifferentiate: if y = xⁿ then dy/dx = nxⁿ⁻¹. Find first and second derivatives
5.2bOFFICIALAssociate derivative with slope and tangent line
5.2bSTUDENTKnow that the derivative gives the slope of the tangent at any point
5.2cOFFICIALApply differentiation to rates of change
5.2cSTUDENTUse derivatives to find how fast things change (speed, growth rate etc.)
5.2dOFFICIALFind maxima and minima
5.2dSTUDENTFind turning points: set dy/dx = 0 and solve; use second derivative to classify
5.2eOFFICIALSketch curves using derivatives
5.2eSTUDENTUse calculus to help sketch graphs: find turning points, increasing/decreasing regions
