Complex Numbers

Leaving Certificate Ordinary Level

Algebra, Geometry, Conjugates, Modulus & Division

ℂ = { a + bi | a, b ∈ ℝ, i² = −1 }

Every real number is complex: a = a + 0i
Every imaginary number: 0 + bi

Powers of i

The powers of i repeat in a cycle of 4:

i⁰ = 1 i¹ = i i² = −1 i³ = −i

Q1. Simplify these powers of i:

(a) i⁰
(b) i¹
(c) i²
(d) i³

(a) i⁰ = 1

(b) i¹ = i

(c) i² = −1

(d) i³ = i² × i = −1 × i = −i

Powers of i — Modular Arithmetic

To find in: divide n by 4, use the remainder

Q2. Simplify using modular arithmetic:

(a) i²⁰
(b) i⁻⁷
(c) i³³
(d) i⁻¹⁸
(a) 20 ÷ 4 = 5 remainder 0
∴ i²⁰ = i⁰ = 1
(b) −7 mod 4 = −7 + 8 = 1
∴ i⁻⁷ = i¹ = i
(c) 33 ÷ 4 = 8 remainder 1
∴ i³³ = i¹ = i
(d) −18 mod 4 = −18 + 20 = 2
∴ i⁻¹⁸ = i² = −1

Scaling Complex Numbers

k(a + bi) = ka + kbi — changes magnitude, not argument

Q4. Scale each complex number:

(a) 2(3 + 4i)
(b) −3(2 − i)
(c) ½(6 + 8i)
(a) 2×3 + 2×4i
= 6 + 8i
(b) −3×2 + (−3)×(−i)
= −6 + 3i
(c) ½×6 + ½×8i
= 3 + 4i

Products of Complex Numbers

(a + bi)(c + di) = (ac − bd) + (ad + bc)i

Conjugates

If z = a + bi, then z̄ = a − bi

Flip the sign of the imaginary part

Q7. Find each conjugate:

(a) 3 + 4i
(b) 5 − 2i
(c) −6 + 7i
(d) 2 − 5i

(a) 3 − 4i

(b) 5 + 2i

(c) −6 − 7i

(d) 2 + 5i

Modulus

|z| = |a + bi| = √(a² + b²)

Distance from origin • Always non-negative • Signs don't matter

Q8. Find the modulus:

(a) |3 + 4i|
(b) |4 + 3i|
(c) |4 − 3i|
(a) √(3² + 4²) = √(9 + 16) = √25 = 5
(b) √(4² + 3²) = √(16 + 9) = √25 = 5
(c) √(4² + (−3)²) = √(16 + 9) = √25 = 5
Notice: all three give the same answer!

The Product zz̄

zz̄ = (a + bi)(a − bi) = a² + b² = |z|²

Always a real number!

Q9. Compute each product:

(a) 3² + 4² = 9 + 16 = 25
(b) 5² + 2² = 25 + 4 = 29

Division of Complex Numbers

Multiply top & bottom by conjugate of denominator

Q10. Divide and simplify:

(a) 6i ÷ 3i
(b) 8 ÷ 2i
(a) 6i / 3i = 6/3 = 2
(b) 8 / 2i × (−i)/(−i)
= −8i / (−2i²)
= −8i / 2 = −4i

Division — More Practice

Q10 cont.

(c) (6 + 8i) ÷ 2
