Ending Digits

A square number can only end in:

0, 1, 4, 5, 6, 9

So any number ending in 2, 3, 7, or 8 is never a square.

Parity (Odd / Even)

Claim: Odd² is odd, Even² is even.

Even number: 2n → (2n)² = 4n² (even)

Odd number: 2n + 1 → (2n + 1)² = 4n² + 4n + 1 (odd)

Building Squares Quickly

If you know one square, you can get the next by adding:

current base + next base

Example:

20² = 400
21² = 400 + 20 + 21 = 441
